# Reversing:
STL Reverse function:
```cpp
std::reverse(x.begin(), x.end());
```
- It is **inplace** reversal
Where x can be :
1. String
2. Vector
3. Array  
etc

> Time Complexity: O(N)  
> Auxiliary Space: O(1)  



### M2: Using the Constructor for STRING reversal 
```cpp
string rev = string(str.rbegin(), str.rend());
```
> Time Complexity: O(N)
> Auxiliary Space: O(1)
